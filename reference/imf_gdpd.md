# IMF GDP Deflator Values

A dataset containing Gross Domestic Product Price Deflator index values
for various countries and years.

## Usage

``` r
imf_gdpd
```

## Format

A tibble with 9,292 rows and 3 variables:

- COUNTRY:

  Country name - WEO format

- YEAR:

  Year

- NGDP_D:

  Gross Domestic Product Price Deflator index
