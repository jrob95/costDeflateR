# IMF Purchasing Power Parity Values

A dataset containing purchasing power parity values for various
countries and years from the IMF.

## Usage

``` r
imf_ppp
```

## Format

A tibble with 9,009 rows and 3 variables:

- COUNTRY:

  Country name - WEO format

- YEAR:

  Year

- PPPEX:

  Purchasing power parity value
