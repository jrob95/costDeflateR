# Roberts et al., 2023 test Data

A dataset containing test data for reference and testing.

## Usage

``` r
rob_sr
```

## Format

A tibble with 23 rows and 5 variables:

- Reference:

  Reference of study

- Field:

  Name of field/cost

- country_orig:

  Country name for cost

- year_orig:

  Year of cost

- cost_orig:

  Cost
