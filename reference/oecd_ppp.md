# OECD Purchasing Power Parity Values

A dataset containing purchasing power parity values for various
countries and years from the OECD

## Usage

``` r
oecd_ppp
```

## Format

A tibble with 2,079 rows and 3 variables:

- COUNTRY:

  Country name - WEO format

- YEAR:

  Year

- PPP:

  Purchasing power parity value

## Source

\<https://www.oecd.org/en/data/indicators/purchasing-power-parities-ppp.html\>
