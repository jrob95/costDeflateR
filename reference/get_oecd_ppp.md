# Get OECD PPP values

Get Purchasing power parity (PPP) values from OECD data sets.

## Details

Internal function to be used by developers to update internal (i.e.:
redundant OECD PPP values) and also by the user when loading the package
to ensure up to date values are used in the \`deflate()\` function. see
\<a href =
"https://www.oecd.org/en/data/indicators/purchasing-power-parities-ppp.html"\>
OECD website\</a\> for more information.
